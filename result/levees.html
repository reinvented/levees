This is the 21st year I've been collating and confirming this information. If you're new to all of this and want to give it a try, read How to Levee.

The list below is incomplete; I'm still gathering information from organizers. All levees listed below have been confirmed with organizers: if you have changes to the information below, or know of other levees, please drop me a line.
